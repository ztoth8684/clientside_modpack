# Basic Setup
Download the Fabric Launcher at https://fabricmc.net/use/installer/

Run it and select the current server version to create a profile for.


# Windows
Place all .jar files in your `C:\Users\<usr>\AppData\Roaming\.minecraft\mods` folder.

# Mac
Place all .jar files in your `~/Library/Application Support/minecraft/Mods` folder.


# Included Mods
## APIs
These mods are just scaffolding to allow others to work.

[Fabric API](https://modrinth.com/mod/fabric-api), [MaLiLib](https://modrinth.com/mod/malilib), [Cloth Config](https://modrinth.com/mod/cloth-config) [Voxy Server](https://modrinth.com/mod/voxyserver)

## Performance
These mods make the game run smoother.

[FerriteCore](https://modrinth.com/mod/ferrite-core) — reduces memory usage

[Sodium](https://modrinth.com/mod/sodium) — rendering overhaul to improve frame rates

## Simple Additions
These mods add things without any configuration.

[Mod Menu](https://modrinth.com/mod/modmenu) — lets you see what mods you have installed

[Voxy](https://modrinth.com/mod/voxy) — renders distant terrain so that you can see further without a high render distance

[Locator Heads](https://modrinth.com/mod/locator-heads) — shows player heads on the locator bar instead of colored dots

[Voice Chat](https://modrinth.com/plugin/simple-voice-chat) — allows proximity & global voice chat directly through the Minecraft server

## Tunable Additions
These mods add things that are more configurable/technical.

[MiniHUD](https://modrinth.com/mod/minihud) — tweakable improvements to the HUD, including adding info lines to the screen and being able to see structure bounding boxes

[Tweakeroo](https://modrinth.com/mod/tweakeroo) — client-side tweaks including zoom, auto-clickers, hand restock, a free camera, and more

[Litematica](https://modrinth.com/mod/litematica) — schematic renderer that allows you to design things in a different world then make a schematic to help you rebuild them block-by-block in survival
